# Ekiti State Infrastructure Dashboard

Interactive GIS-based dashboard for the **Inventory and Mapping of Community-Based Infrastructural Facilities** across all 16 Local Government Areas of Ekiti State, Nigeria.

## Live Dashboard

**[View the Dashboard](https://YOUR_USERNAME.github.io/ekiti-state-dashboard/)**

*(Replace YOUR_USERNAME with your GitHub username after deployment)*

## What's Inside

- **5,183 geotagged facilities** across 14 sectors (Education, Health, Water, Electricity, Roads, Banks, etc.)
- **Interactive map** with filtering by sector, LGA, and ownership type
- **Multi-criteria community scoring** — weight different infrastructure factors to rank communities
- **Facility photo popups** — click any point to see details and field photographs
- **Needs Assessment** — survey data from 3,243 respondents across all 16 LGAs, with gender breakdown and priority ranking
- **State-level overview** with comparative charts and KPIs

## Project Background

This project was conceived as a need-driven research to undertake a comprehensive spatial and attribute inventory, documentation, and mapping of existing infrastructures and facilities within Ekiti State. The objective is to develop an integrated GIS database for decision support, infrastructure maintenance, and future planning.

**Project Consultant:** Prof. Peter O. Adeniyi  
**Developed by:** Polad Technologies Ltd  
**Data Collection:** Field survey with GPS, completed September 2010  
**Communities Surveyed:** 451 across 16 LGAs  
**Survey Respondents:** 3,243

## Technology

Single self-contained HTML file using:
- **Leaflet.js** — interactive mapping
- **Chart.js** — data visualizations
- **CartoDB Voyager** — base map tiles

No server required. Works in any modern browser.
